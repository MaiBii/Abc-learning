ABC Learning – English Course Landing Page
英語学習コース・ランディングページ

## 1. Overview 概要

This project is a front‑end landing page for an online English‑learning platform. It showcases UI layout, responsive design, and component‑based structuring using HTML and CSS. All course information and images are sample data for design demonstration.  
本プロジェクトは、オンライン英語学習プラットフォームのランディングページです。HTML と CSS を使用し、UI レイアウト、レスポンシブデザイン、コンポーネント構成を実装しています。コース情報や画像はデザイン用のサンプルデータです。

## 2. Features 主な機能

- Hero section with call‑to‑action / CTA 付きヒーローセクション
- Course list with hover effects / ホバー効果付きコース一覧
- Student feedback section / 受講者レビュー表示
- Sticky header navigation / 固定ヘッダー・ナビゲーション
- Responsive layout using Flexbox / Flexbox を使ったレスポンシブレイアウト
- Clean folder structure for assets / 整理されたアセット構成

## 3. Tech Stack 使用技術

- HTML
- CSS
- Google Fonts (Montserrat)

## 4. System Flow システムフロー

- User visits the landing page / ユーザーがページにアクセス
- UI displays hero, courses, and feedback sections / UI がヒーロー・コース・レビューを表示
- Buttons and navigation guide users to actions / ボタンとナビゲーションでユーザーを誘導
- Layout adjusts based on screen size / 画面サイズに応じてレイアウトが調整

## 5. Page Sections ページ構成

- Header (logo + navigation) / ヘッダー（ロゴ＋ナビ）
- Hero banner with image and CTA / ヒーローバナー（画像＋CTA）
- Popular courses / 人気コース一覧
- Student feedback / 受講者レビュー
- Footer with company info / フッター（会社情報）

## 6. How to Run 実行方法

- Download the project folder / プロジェクトフォルダをダウンロード
- Open `index.html` in a browser / ブラウザで `index.html` を開く
